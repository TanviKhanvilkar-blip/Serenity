# Serenity
Mental Health Help Website

## Overview

Serenity is a mental wellness web application developed during a short-format hackathon competition. The platform aims to provide users with simple tools for self-reflection, mood tracking, and emotional support through an intuitive and accessible interface.

## Features

### Home

* Introduction to the platform and its purpose.

### Support

* Quick access to coping strategy categories and wellness resources.

### Journaling

* Write and save personal journal entries during the active session.

### Mood Tracker

* Record moods with timestamps and dates to monitor emotional patterns.

### AI Therapist

* Integrated Botpress chatbot configured to provide supportive conversations and basic mental wellness guidance.
